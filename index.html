基于GitBook的博客搭建

  • 通过NPM安装gitbook
    npm install gitbook-cli -g
  • 初始化项目(新建文件)
    gitbook init
  • 启动服务(http://localhost:4000/)
    gitbook serve
  • 根目录下新建book.json
    {

  • 安装插件
    gitbook install
  • 关联git仓库
  • 全局安装gh-pages工具
    npm install gh-pages -g
  • _book部署到分支
    gh-pages -d _book
  • 关联域名，添加CNNAME文件
