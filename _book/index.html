1. 基于GitBook的博客搭建

1.1. 通过NPM安装gitbook

npm install gitbook-cli -g

1.2. 初始化项目(新建文件)

gitbook init

1.3. 启动服务

gitbook serve
// serve on http://localhost:4000/
